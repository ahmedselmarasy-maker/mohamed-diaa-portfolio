# بورتوفوليو محمد ضياء - مصمم جرافيك

بورتوفوليو احترافي لمصمم الجرافيك محمد ضياء، يعرض خبراته وأعماله ومهاراته.

## المميزات

- ✅ تصميم عصري واحترافي
- ✅ متجاوب مع جميع الأجهزة (Responsive)
- ✅ تأثيرات بصرية سلسة
- ✅ أقسام منظمة:
  - قسم Hero مع الصورة والتحية
  - نبذة عن محمد ضياء
  - الأعمال والمشاريع
  - المهارات (تقنية، عملية، و Soft Skills)
  - قسم التواصل

## كيفية الاستخدام

1. افتح ملف `index.html` في المتصفح
2. يمكنك تعديل المحتوى في ملف `index.html`
3. يمكنك تعديل الألوان والتصميم في ملف `style.css`
4. يمكنك إضافة المزيد من التفاعلية في ملف `script.js`

## ملاحظات

- يمكنك إضافة صورة شخصية حقيقية في قسم Hero عن طريق استبدال `profile-image-placeholder` بصورتك
- يمكنك تحديث معلومات التواصل في قسم Contact
- يمكنك إضافة صور لأعمالك في قسم Works

## التقنيات المستخدمة

- HTML5
- CSS3 (مع CSS Variables و Animations)
- JavaScript (Vanilla JS)
- Google Fonts (Cairo)
